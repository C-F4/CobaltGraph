â“ FREQUENTLY ASKED QUESTIONS:

Q: Does bash work on Windows?
A: No, bash is NOT native to Windows. But bash DOES work in:
   - WSL (Windows Subsystem for Linux) âœ…
   - Git Bash âœ…
   - Cygwin âœ…
   That's why we provide cobaltgraph.bat for native Windows.

Q: Which launcher should I use?
A: Use "python cobaltgraph.py" - it works everywhere.

Q: What's the difference between device mode and network mode?
A: Device mode monitors only YOUR computer's connections (no admin needed).
   Network mode monitors ALL devices on your network (requires admin/sudo).

Q: Can I use the bash scripts on Windows?
A: Only if you're in WSL, Git Bash, or Cygwin. For native Windows CMD/PowerShell,
   use cobaltgraph.bat or python cobaltgraph.py instead.

Q: Why are there symlinks in the root?
A: For convenience! Instead of typing "python bin/cobaltgraph.py", you can just type
   "python cobaltgraph.py". Both work the same.
